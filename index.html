bin

Useful scripts for your ~/bin directory.

Install

Clone the repo, preferably to ~/bin:

git clone git@github.com:adamatan/bin.git ~/bin

Add the bin path to your .bashrc or .zshrc, e.g.:

PATH=$PATH:/home/adam/bin
